# GrowthOS v0.6.0 Scope

## Release Name

GrowthOS v0.6.0 - Publishing Engine Foundation

## In Scope

- Draft persistence.
- Draft lifecycle and state transitions.
- Draft version history.
- Duplicate, archive, restore, soft delete.
- Version-pinned scheduling and publishing jobs.
- Workspace-scoped ownership.
- Brand and platform association.
- Review workflow.
- Content Library.
- Calendar and scheduling foundation.
- Publishing queue foundation.
- Deterministic mock publishing adapter.
- External platform adapter interfaces.
- Publishing screens and role-aware UI.
- Backend, frontend, Playwright, database, and security validation.

## Out Of Scope

- Real OAuth integrations.
- Real platform publishing.
- Browser automation for social posting.
- Paid APIs.
- Billing.
- Analytics Hub.
- Workflow Automation.
- Website Intelligence.
- Lead Intelligence.
- MFA.
- Admin portal.
- HttpOnly-cookie migration.

## Acceptance Criteria

- Editors can create and submit drafts.
- Admin/owner can approve and schedule.
- Viewer cannot write.
- Mock publishing can succeed, fail transiently, fail permanently, and retry.
- Scheduled and queued jobs publish the approved version that was pinned at scheduling/enqueue time.
- Calendar and queue views work on desktop and mobile.
- Existing modules do not regress.

## Release Gate

- Backend tests pass.
- PostgreSQL migration passes.
- Frontend lint passes.
- Frontend build passes.
- Playwright smoke passes.
- Security review passes.
- No forbidden artifacts or secrets staged.
