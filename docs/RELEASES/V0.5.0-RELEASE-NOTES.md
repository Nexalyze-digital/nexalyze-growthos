# GrowthOS v0.5.0 - Platform Identity, Workspaces, And Database

## Highlights

- Authentication foundation with registration, login, logout, access tokens, and refresh tokens.
- Password hashing with PBKDF2-HMAC-SHA256.
- Organizations, workspaces, memberships, and role-based permissions.
- Workspace switcher in the GrowthOS shell.
- Protected frontend dashboard.
- Workspace-scoped Brand Brain.
- Workspace-scoped Research Hub.
- Workspace-scoped AI Content Studio context lookup.
- SQLAlchemy 2.x database layer.
- Alembic migration scaffolding.
- SQLite-supported local and test configuration.
- PostgreSQL production configuration path.
- JSON migration utility with dry-run and live modes.
- Backup before live JSON migration.
- Audit events for authentication and data changes.
- Security headers and restricted CORS headers/methods.

## Known Limitations

- No MFA.
- No email verification.
- No password reset workflow.
- No admin portal.
- No production PostgreSQL credentials are committed.
- JSON migration requires an existing workspace id.
- Playwright smoke remains local-script based.
