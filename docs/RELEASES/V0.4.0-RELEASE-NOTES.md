# GrowthOS v0.4.0 - Research Hub

## Highlights

- Research Hub workspace for structured AI-generated market intelligence.
- Persistent research runs with history, retrieve, regenerate, and delete workflows.
- FastAPI research schemas, routes, service layer, repository layer, and provider abstraction.
- Ollama research provider for local AI synthesis.
- Deterministic mock provider for repeatable demos, tests, and fallback.
- Automatic mock fallback when Ollama is unavailable or returns invalid output.
- Brand Brain context injection into research prompts.
- Structured summaries, key findings, opportunities, risks, recommendations, follow-up questions, and source notes.
- Source-integrity safeguards that disclose synthesis-only behavior.
- Responsive desktop and mobile Research Hub experience.
- Playwright smoke coverage for release-critical flows.

## Known Limitations

- No live web crawling.
- No source fetching or citation verification.
- No external search APIs.
- JSON persistence remains local and single-user.
- No authentication or workspace ownership.
- No database persistence.
- No vector search.

## Scope Note

Research Hub v0.4.0 performs local AI synthesis only. Live crawling, source fetching, verified-source research, search integrations, database persistence, and workspace ownership are deferred to later releases.
