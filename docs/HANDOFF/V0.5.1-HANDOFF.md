# GrowthOS v0.5.1 Handoff

## Summary

v0.5.1 validates and hardens the v0.5.0 platform foundation before Publishing Engine work begins.

## Implemented

- PostgreSQL driver support.
- SQLAlchemy pool pre-ping and pool configuration.
- PostgreSQL migration validation.
- PostgreSQL backend test validation.
- Backup and restore scripts.
- Frontend access-token refresh.
- Refresh concurrency protection.
- Role-aware frontend write-control hiding.
- CI-ready Playwright workflow.
- Security and deployment documentation.

## Startup

Backend:

```powershell
.\scripts\run-api.ps1
```

Frontend:

```powershell
.\scripts\run-web.ps1
```

PostgreSQL validation:

```powershell
$env:DATABASE_URL="postgresql+psycopg://postgres@localhost:5432/growthos_v051_validation"
```
